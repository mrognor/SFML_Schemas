Файл Nodes.txt содержит путь, название, состояние открытости/закрытости если это папка
переменную для определения типа элемента(папка или нода),
Название связано с файлом в котором лежит определение ноды

Пример для And
ProjectName/BaseNodes/And/ And false false


Файлы ноды содержат в себе:
Строка входов. По порядку и через слешы
Строка выходов. По порядку и через слешы
Строка функций. По порядку и через слешы. В начале имя выхода, потом ":", потом формула, потом слеш

Пример для And
InputLine: A/B
OutputLine: A
FormulaLine: A:and(A,B)
